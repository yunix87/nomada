# Hola Mundo en C++

Este pequeño proyecto contiene un programa en C++ que imprime "Hola, mundo!" en la consola.

Archivos:

- `main.cpp` — programa fuente.

Compilar y ejecutar en Windows (PowerShell):

Usando g++ (MinGW / MSYS2):

```powershell
g++ -o hola main.cpp
./hola
```

Usando el compilador de Microsoft (cl.exe) desde el Developer Command Prompt:

```powershell
cl /EHsc main.cpp
main.exe
```
